# Asish Portfolio

Professional software engineering portfolio focused on backend systems, scalable applications, and algorithmic engineering projects.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Features

- Responsive engineering-focused UI
- Recruiter-friendly project showcase
- Resume preview integration
- GitHub and LinkedIn integration
- Smooth scrolling navigation
- Modular React component structure

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
